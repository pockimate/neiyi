## 🎯 现在部署到 Vercel（只需 3 步）

### 第 1 步：访问 Vercel

打开浏览器，访问：
```
https://vercel.com
```

点击右上角 **"Sign Up"**（如果已有账号点 "Login"）

### 第 2 步：用 GitHub 登录

1. 选择 **"Continue with GitHub"**
2. 点击 **"Authorize Vercel"** 授权
3. 登录成功后进入仪表板

### 第 3 步：导入并部署

1. 点击 **"Add New..."** → **"Project"**

2. 在项目列表找到 **"neiyi"** 仓库
   - 如果没看到，点击 "Adjust GitHub App Permissions" 授权

3. 点击 **"Import"** 按钮

4. 配置页面保持默认设置：
   ```
   Project Name: neiyi
   Framework Preset: Other
   Root Directory: ./
   Build Command: [留空]
   Output Directory: [留空]
   ```

5. 点击蓝色大按钮 **"Deploy"**

6. 等待 30-60 秒... ⏳

7. **完成！** 🎉

## 🆘 遇到问题？

### 找不到仓库
- 点击 "Adjust GitHub App Permissions"
- 授权 Vercel 访问所有仓库

### 部署失败
- 查看部署日志
- 检查 vercel.json 配置
- 确认所有文件已推送

### 页面 404
- 访问完整路径：`/index.html`
- 等待 2-3 分钟
- 清除浏览器缓存
