## 🎨 核心色彩定义

### 品牌主色 - 玫瑰粉（记忆点）

## 🎯 新增CSS组件

### 按钮系统
```css
.btn-primary    /* 玫瑰粉 - 品牌按钮 */
.btn-accent     /* 优雅金 - CTA按钮 */
.btn-secondary  /* 浅玫瑰粉 - 次要按钮 */
.btn-outline    /* 轮廓按钮 */
```

### 徽章系统
```css
.badge-sale     /* 优雅金 - Sale标签 */
.badge-new      /* 玫瑰粉 - New标签 */
.badge-hot      /* 深玫瑰粉 - Hot标签 */
```

### 视觉效果
```css
.glow-primary   /* 品牌色光晕 */
.glow-accent    /* 金色光晕 */
.text-gradient  /* 渐变文字 */
.gradient-brand /* 品牌渐变背景 */
```
