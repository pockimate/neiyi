- 必须同意服务条款

- 粉色背景

- 点击展开用户菜单

### 用户下拉菜单

1. **用户信息**
   - 显示姓名
   - 显示邮箱

2. **菜单项**
   - 👤 My Profile
   - 🛍️ My Orders
   - ❤️ Wishlist
   - ⚙️ Settings

3. **退出登录**
   - 🚪 Sign Out（红色）

## 🎯 用户流程

### 注册流程
1. 点击导航栏 "Sign In"
2. 点击 "Sign up"
3. 填写注册表单
4. 同意服务条款
5. 提交注册
6. 自动登录并跳转首页

### 登录流程
1. 点击导航栏 "Sign In"
2. 输入邮箱和密码
3. 可选：勾选"记住我"
4. 提交登录
5. 跳转到之前页面或首页

### 退出流程
1. 点击用户头像
2. 点击 "Sign Out"
3. 清除登录数据
4. 跳转到首页
